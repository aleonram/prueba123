Git & GitHub Workshop

Workshop Overview

Welcome to the Git and GitHub Workshop! In this session, we will learn about version control using Git and how to collaborate using GitHub. Whether you're new to Git or looking to expand your skills, this workshop is for you!

Workshop Objectives

  • Introduction to Git and GitHub: Understand what Git is, how it works, and how GitHub helps developers collaborate.
  • Introduction to Branches: Learn how to work with Git branches for parallel development and collaboration.
  • Learning Branch Protection Rules: Understand how to set up and manage branch protection rules to maintain code quality and prevent mistakes.
  • Getting Familiar with GitHub Copilot and Codespaces: Explore GitHub Copilot for AI-assisted coding and GitHub Codespaces for development environments in the cloud.
  • Understanding GitHub Gists, Pages, and Issues: Learn about GitHub Gists for sharing code snippets, GitHub Pages for hosting static websites, and GitHub Issues for tracking tasks.
  • Introduction to a GUI for Git: Get hands-on experience with graphical user interfaces for Git
